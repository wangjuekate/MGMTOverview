Drawing on the board

Draw the Technology S Curve and Consumer Adoption S Curve for IBM's software and cloud services

CEO

Should shift towards cloud service?

Takeaway for CEO

Tech S curve shift from large entities to small entities

Demand S curve shift from more technical experts to general public

CTO

Question: What technology should IBM develop more for year 2011?


A: More research funding needs to be put in developing software

B: More research funding needs to be put in developing cloud services

CTO Takeaways

Approximately two-thirds of external software segment revenue is annuity-based, coming from recurring license charges and ongoing subscription and support from one-time charge (OTC) arrangements.

Consumer adoption S curve indicates that public would be more accepting cloud services.

CMO

Question: How IBM's marketing budget should be spent for year 2011?


A: More budget should be put on advertising software

B: More budget should be put on advertising cloud services

CMO Takeaways

Market positioning: Emphasize IBM's integrated capabilities - services, software, systems, and research combined to provide business solutions

Revenue model advantage: 60% of Global Services revenue is annuity-based (recurring), making cloud services a sustainable growth strategy

COO

Question: How IBM's arrange the operation for year 2011?


A: IBM would partner more for cloud infrastructure

B: IBM should try develop cloud infrastructure solo

COO Takeaways

Rationale: Cloud initiatives tailored to particular industries work best in conjunction with partners and clients

Competitive advantage: Leverage existing software expertise and IP portfolio while building cloud capabilities through partnerships

CHRO

Question: Who should IBM hire for year 2011?


A: IBM should hire more remote workers globally

B: IBM should hire more employees from local regions

CHRO Takeaways

Talent gap: Current leadership has 3 software group managers but no cloud service top manager

Cloud expertise needed: Prioritize hiring cloud service professionals

Global workforce: Cloud infrastructure is global by nature - physical presence is not always necessary

CFO

Question: How should IBM divide the earned revenue for year 2011?


A: 50% for software development 50% for cloud infrastructure development

B: 20% for software development 80% for cloud infrastructure development

CFO Takeaways

Current revenue: Software generates $700M net revenue vs. Cloud $200M net revenue

Future growth: Cloud applications bring more service opportunities despite lower current revenue

Strategic balance: Software helps stabilize current revenue while cloud investment drives future growth
